## 💎 Pro Tips & Best Practices

### 1. **Always Start with EDA**
```python
# BEFORE engineering
df.describe()
df.corr()
df.hist(figsize=(15,12))

# Understand THEN engineer
```

### 2. **Document Your Features**
```python
# BAD
df['f1'] = df['a'] / df['b']

# GOOD
# price_per_sqft: Captures cost efficiency
# Hypothesis: Smaller homes cost more per sqft
# Expected: Positive correlation with quality ratings
df['price_per_sqft'] = df['price'] / df['sqft']
```

### 3. **Version Your Feature Sets**
```python
features_v1 = ['age', 'income']  # Baseline: 70%
features_v2 = features_v1 + ['income_log']  # v2: 75%
features_v3 = features_v2 + ['age_income_ratio']  # v3: 82% ✓
```

### 4. **Validate Everything**
```python
# Split BEFORE engineering
X_train, X_test, y_train, y_test = train_test_split(X, y)

# Engineer on training
X_train_eng = engineer_features(X_train)

# Apply to test
X_test_eng = engineer_features(X_test)

# Compare performance
```

### 5. **Use Pipelines for Production**
```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.decomposition import PCA

pipeline = Pipeline([
    ('scaler', StandardScaler()),
    ('pca', PCA(n_components=10)),
    ('model', RandomForestClassifier())
])

# One fit, one predict - no leakage!
pipeline.fit(X_train, y_train)
predictions = pipeline.predict(X_test)
```

## 🔍 Common Pitfalls to Avoid

### ❌ **Data Leakage**
```python
# WRONG - Test data influenced scaling!
scaler.fit(entire_dataset)

# RIGHT - Only learn from training
scaler.fit(X_train)
```

### ❌ **Creating Too Many Features**
```python
# WRONG - 1000 random combinations
for f1 in features:
    for f2 in features:
        df[f'{f1}_{f2}'] = df[f1] * df[f2]

# RIGHT - Meaningful combinations only
df['price_per_sqft'] = df['price'] / df['sqft']
```

### ❌ **Not Validating**
```python
# WRONG - Engineer blindly
add_100_features()

# RIGHT - Test each addition
baseline_score = 0.80
add_feature()
new_score = 0.82  # Improvement? Keep it!
```

### ❌ **Ignoring Domain Knowledge**
```python
# WRONG - Generic features
df['ratio_1'] = df['col1'] / df['col2']

# RIGHT - Meaningful features
df['debt_to_income_ratio'] = df['debt'] / df['income']
```

### ❌ **Forgetting Production**
```python
# WRONG - Can't reproduce in production
df['new_feature'] = some_complex_function()

# RIGHT - Save transformation
transformer = FunctionTransformer(some_complex_function)
joblib.dump(transformer, 'transformer.pkl')
```
