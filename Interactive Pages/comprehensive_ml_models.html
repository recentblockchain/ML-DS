📈 Supervised Learning - Regression 12 models ▼
Linear Regression
Regression Simple
Fits a straight line (or hyperplane) through data points to predict continuous values. Uses least squares method.
📋 Use Cases:
  • House price prediction
  • Sales forecasting
  • Trend analysis
Polynomial Regression
Regression Simple
Extension of linear regression that fits polynomial curves (quadratic, cubic, etc.) to capture non-linear relationships.
📋 Use Cases:
  • Growth rate modeling
  • Disease progression
  • Population dynamics
Ridge Regression (L2)
Regression Medium
Linear regression with L2 regularization to prevent overfitting by penalizing large coefficients.
📋 Use Cases:
  • High-dimensional data
  • Multicollinearity handling
  • Gene expression analysis
Lasso Regression (L1)
Regression Medium
Linear regression with L1 regularization that can reduce some coefficients to zero, performing feature selection.
📋 Use Cases:
  • Feature selection
  • Sparse models
  • Compressed sensing
Elastic Net
Regression Medium
Combines L1 and L2 regularization, balancing feature selection and coefficient shrinkage.
📋 Use Cases:
  • Grouped features
  • When Ridge & Lasso alone aren't enough
  • Genomics data
Support Vector Regression (SVR)
Regression Medium
Uses SVM principles for regression, finding a hyperplane that fits data within a margin of tolerance.
📋 Use Cases:
  • Stock price prediction
  • Energy consumption forecasting
  • Non-linear relationships
Decision Tree Regression
Regression Medium
Creates a tree structure to partition feature space and predict values at leaf nodes.
📋 Use Cases:
  • Non-linear patterns
  • Mixed feature types
  • Interpretable predictions
Random Forest Regression
Regression Medium
Ensemble of decision trees that averages predictions to reduce overfitting and improve accuracy.
📋 Use Cases:
  • Real estate appraisal
  • Demand forecasting
  • Feature importance ranking
Gradient Boosting Regression
Regression Advanced
Builds trees sequentially, each correcting errors of the previous one. Includes XGBoost, LightGBM, CatBoost.
📋 Use Cases:
  • Kaggle competitions
  • Complex relationships
  • High-accuracy requirements
K-Nearest Neighbors Regression
Regression Medium
Predicts by averaging the values of K nearest neighbors in the feature space.
📋 Use Cases:
  • Recommendation systems
  • Small datasets
  • Local patterns
Neural Network Regression
Regression Advanced
Multi-layer networks that learn complex non-linear mappings from inputs to continuous outputs.
📋 Use Cases:
  • Image-based predictions
  • Time series with complex patterns
  • Multi-modal data
Bayesian Ridge Regression
Regression Medium
Probabilistic approach to linear regression that provides uncertainty estimates in predictions.
📋 Use Cases:
  • Uncertainty quantification
  • Small datasets
  • Medical predictions
🎯 Supervised Learning - Classification 15 models ▼
Logistic Regression
Classification Simple
Binary classification using sigmoid function to estimate probability of class membership.
📋 Use Cases:
  • Spam detection
  • Disease diagnosis (yes/no)
  • Customer churn prediction
K-Nearest Neighbors (KNN)
Classification Medium
Classifies based on majority vote of K nearest neighbors. Simple but effective.
📋 Use Cases:
  • Pattern recognition
  • Recommendation systems
  • Image classification
Support Vector Machine (SVM)
Classification Medium
Finds optimal hyperplane that maximizes margin between classes. Works with kernel trick for non-linear data.
📋 Use Cases:
  • Text classification
  • Handwriting recognition
  • Bioinformatics
Naive Bayes
Classification Simple
Probabilistic classifier based on Bayes' theorem with "naive" independence assumption.
📋 Use Cases:
  • Spam filtering
  • Sentiment analysis
  • Document categorization
Decision Tree Classifier
Classification Medium
Tree-based model that makes decisions through series of if-then rules based on feature values.
📋 Use Cases:
  • Loan approval
  • Medical diagnosis
  • Customer segmentation
Random Forest Classifier
Classification Medium
Ensemble of decision trees using bagging and feature randomness for robust predictions.
📋 Use Cases:
  • Fraud detection
  • Disease prediction
  • Credit scoring
Gradient Boosting Classifier
Classification Advanced
Sequential ensemble method where each tree corrects errors of previous ones. Includes XGBoost, LightGBM, CatBoost.
📋 Use Cases:
  • Click-through rate prediction
  • Risk assessment
  • Competition-winning models
AdaBoost
Classification Advanced
Adaptive boosting that focuses on misclassified examples by adjusting their weights.
📋 Use Cases:
  • Face detection
  • Imbalanced datasets
  • Weak learner combination
Extra Trees Classifier
Classification Medium
Similar to Random Forest but uses random thresholds for splits, reducing variance further.
📋 Use Cases:
  • Feature importance analysis
  • High-dimensional data
  • Faster training than RF
Perceptron
Classification Simple
Simplest neural network with single layer. Linear classifier for binary problems.
📋 Use Cases:
  • Linearly separable data
  • Online learning
  • Foundation for neural networks
Linear Discriminant Analysis (LDA)
Classification Medium
Finds linear combination of features that best separates classes. Also used for dimensionality reduction.
📋 Use Cases:
  • Face recognition
  • Marketing prediction
  • Medical diagnosis
Quadratic Discriminant Analysis (QDA)
Classification Medium
Extension of LDA that allows for quadratic decision boundaries between classes.
📋 Use Cases:
  • Non-linear class boundaries
  • Different covariance per class
  • Pattern recognition
Neural Network Classifier
Classification Advanced
Multi-layer perceptron (MLP) with hidden layers that learns complex non-linear decision boundaries.
📋 Use Cases:
  • Image classification
  • Speech recognition
  • Complex patterns
Gaussian Process Classifier
Classification Medium
Probabilistic model that provides prediction uncertainty. Non-parametric Bayesian approach.
📋 Use Cases:
  • Small datasets
  • Uncertainty quantification
  • Active learning
Stacking Classifier
Classification Advanced
Meta-ensemble that combines predictions from multiple models using another model as final estimator.
📋 Use Cases:
  • Maximizing accuracy
  • Combining diverse models
  • Kaggle competitions
🔍 Unsupervised Learning - Clustering 10 models ▼
K-Means Clustering
Clustering Simple
Partitions data into K clusters by minimizing within-cluster variance. Fast and scalable.
📋 Use Cases:
  • Customer segmentation
  • Image compression
  • Document clustering
DBSCAN
Clustering Medium
Density-based clustering that finds clusters of arbitrary shape and identifies outliers.
📋 Use Cases:
  • Anomaly detection
  • Spatial data clustering
  • Non-spherical clusters
Hierarchical Clustering
Clustering Medium
Creates tree-like structure (dendrogram) of nested clusters using agglomerative or divisive approach.
📋 Use Cases:
  • Gene expression analysis
  • Taxonomy creation
  • Social network analysis
Gaussian Mixture Model (GMM)
Clustering Medium
Probabilistic model that assumes data is mixture of Gaussian distributions. Soft clustering.
📋 Use Cases:
  • Image segmentation
  • Speaker recognition
  • Anomaly detection
Mean Shift
Clustering Medium
Non-parametric clustering that shifts points toward modes of density function.
📋 Use Cases:
  • Image segmentation
  • Object tracking
  • Computer vision
Spectral Clustering
Clustering Medium
Uses eigenvalues of similarity matrix to reduce dimensions before clustering. Good for graph-based data.
📋 Use Cases:
  • Image segmentation
  • Community detection
  • Non-convex clusters
BIRCH
Clustering Simple
Balanced Iterative Reducing and Clustering using Hierarchies. Memory-efficient for large datasets.
📋 Use Cases:
  • Large datasets
  • Streaming data
  • Data reduction
OPTICS
Clustering Medium
Ordering Points To Identify Clustering Structure. Extension of DBSCAN for varying densities.
📋 Use Cases:
  • Varying density clusters
  • Cluster visualization
  • Outlier detection
Affinity Propagation
Clustering Simple
Identifies exemplars among data points and forms clusters around them through message passing.
📋 Use Cases:
  • Automatic K selection
  • Image clustering
  • Gene discovery
HDBSCAN
Clustering Medium
Hierarchical DBSCAN that extends DBSCAN to varying densities and provides more robust clustering.
📋 Use Cases:
  • Complex cluster shapes
  • Noise robust clustering
  • Hierarchical structure
📉 Dimensionality Reduction 8 models ▼
Principal Component Analysis (PCA)
Dimensionality Reduction Simple
Linear technique that finds orthogonal directions of maximum variance in data.
📋 Use Cases:
  • Data visualization
  • Feature extraction
  • Noise reduction
t-SNE
Dimensionality Reduction Advanced
t-Distributed Stochastic Neighbor Embedding. Non-linear technique excellent for visualization.
📋 Use Cases:
  • High-dimensional visualization
  • Cluster exploration
  • Image embeddings
UMAP
Dimensionality Reduction Advanced
Uniform Manifold Approximation and Projection. Faster than t-SNE with better global structure preservation.
📋 Use Cases:
  • Large dataset visualization
  • Preprocessing for ML
  • Biological data
Autoencoders
Dimensionality Reduction Medium
Neural network that learns compressed representation through encoding-decoding process.
📋 Use Cases:
  • Image compression
  • Anomaly detection
  • Feature learning
Independent Component Analysis (ICA)
Dimensionality Reduction Medium
Separates multivariate signal into independent non-Gaussian components.
📋 Use Cases:
  • Signal separation (cocktail party problem)
  • fMRI analysis
  • Feature extraction
Factor Analysis
Dimensionality Reduction Medium
Statistical method that describes variability among observed variables using fewer latent factors.
📋 Use Cases:
  • Psychometrics
  • Survey analysis
  • Market research
Kernel PCA
Dimensionality Reduction Medium
Non-linear extension of PCA using kernel trick to capture non-linear relationships.
📋 Use Cases:
  • Non-linear dimensionality reduction
  • Denoising
  • Feature extraction
Isomap
Dimensionality Reduction Medium
Non-linear technique that preserves geodesic distances between points on a manifold.
📋 Use Cases:
  • Manifold learning
  • Face recognition
  • Shape analysis

Reinforcement Learning Advanced
